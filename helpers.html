Helpers¶

class InterfaceDef¶

Representation of a predefined interface.

__init__(self, name, prefix, signals)¶
Parameters

  • name – full name of the interface

  • prefix – prefix used as a short identifier for the interface

  • signals ({'required': [str], 'optional': [str]}) – lists of names of required and optional signals in a dict

get_interface_by_name(name: str)¶

Retrieve a predefined interface definition by its name

Returns

InterfaceDef object, or None if there’s no such interface

get_interface_by_prefix(prefix: str)¶

Retrieve a predefined interface definition by its prefix

Returns

InterfaceDef object, or None if there’s no such interface

class Config¶

Configuration class used to store global choices for behavior of Topwrap.

__init__(self, force_interface_compliance=False)¶

A global instance named config is created initially. Users can toggle forcing compliance checks on interfaces:

from fpga_topwrap.config import config

config.force_interface_compliance = False
check_interface_compliance(iface_def, signals)¶
